# Chemistry Minor

Students may earn a Chemistry Minor by completing 25-26 units of Chemistry course work. A minimum GPA of 2.0 is required. To request a minor, please schedule an appointment with Dr. Kathryn McCulloch through CPP Connect.

1. Log in to [CPP Connect](https://nam11.safelinks.protection.outlook.com/?url=https%3A%2F%2Fcpp.campus.eab.com%2Fpal%2Ft-D7TOEf1V&data=05%7C01%7Cchemistry%40cpp.edu%7Cadb8536a9a1d4c621ee708db1680d7c5%7C164ba61e39ec4f5d89ffaa1f00a521b4%7C0%7C0%7C638128514858046682%7CUnknown%7CTWFpbGZsb3d8eyJWIjoiMC4wLjAwMDAiLCJQIjoiV2luMzIiLCJBTiI6Ik1haWwiLCJXVCI6Mn0%3D%7C3000%7C%7C%7C&sdata=TUrqrilWbPIvv1r5slI%2B9er1cjGYWh7tu44HcbT7M4Q%3D&reserved=0 "CPP Connect")
2. Under Care Unit select Advising
3. Under Location select Chemistry Deparment
4. Under Services select Add Minor
5. Select Kathryn McCulloch
6. Scroll to the bottom and under Choose A Time to Meet select a time slot

You can contact Dr. Kathryn McCulloch at kmmcculloch@cpp.edu for more information.

Courses required for the minor are shown below:

| --- | --- |
| CHM 1210   General Chemistry l | 3 |
| CHM 1210L  General Chemistry l Laboratory | 1 |
| CHM 1220   General Chemistry ll | 3 |
| CHM 1220L General Chemistry ll Laboratory | 1 |
| **Analytical Chemistry** | |
| CHM 2210   Quantitative Analysis | 2 |
| CHM 2210L Quantitative Analysis Laboratory | 2 |
| **Organic Chemistry** | |
| CHM 2010 Elements of Organic Chemistry (3)\* OR | 3 or 4 |
| CHM 3140 Organic Chemistry l (4)\* |  |
| CHM 2010L Elements of Organic Chemistry Laboratory OR | 1 |
| CHM 3140L Organic Chemistry l Laboratory |  |
| **Physical Chemistry** | |
| CHM 3010 Modeling the Fundamentals of Physical Chemistry (GE B5)  OR |  |
| CHM 3040 Elements of Physical Chemistry  OR |  |
| CHM 3110 Classical Physical Chemistry |  |
| **Upper Division Chemistry Electives** | |
| Select any 3000-4000 level CHM Courses (except CHM 3140/3140L) to count here | 6 |
| **Total Units** | 25 or 26 |

\*Students who take CHM 3140 will complete the minor in 26 units, those who take CHM 2010 will complete it in 25 units.
