# R Workshop #1: Introduction to R

## Introduction to R

Presented by Jarrod Griffin, Junior Marketing Scientist.
We cover how to install R and RStudio, the layout of RStudio, creating, running, and saving a R script, downloading packages, documenting a R script in RMarkdown, and the various resources available in the R community and its ecosystem.

[If you haven't installed R or RStudio, click here to download on macOS or Windows!](https://www.cpp.edu/cba/customer-insights-lab/news/event/install-r.shtml)

**Learning Outcomes**

1. Understand how to use online resources and community for R and RStudio.
2. Install R and RStudio.
3. Understand the layout and menus of RStudio.
4. Start, Run and Save an R script.
5. Install R packages.
6. Use R Markdown to document a project.
