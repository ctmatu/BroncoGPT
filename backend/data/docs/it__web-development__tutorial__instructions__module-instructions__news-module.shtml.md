# News Module Instructions

## Introduction

The news module is not available by default. If a college or department want a news module on their website, please submit a ticket requesting the news module.

[submit a ticket](https://cpp.service-now.com/ehelp?id=sc_cat_item&sys_id=0f7a475ddbdd9010e00f785e0f96198c&sysparm_category=d571b2f26f84820025ce9e4eae3ee482)

There are two types of news modules:

1. A news feed from the Poly Centric.
2. A custom news feed specific to the website built internally by the department.

This tutorial will go over how to create and set up a custom news module.
