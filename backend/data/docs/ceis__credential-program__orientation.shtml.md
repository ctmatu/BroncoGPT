# General Orientation

This orientation allows one the opportunity to learn more about the Teacher Credential Programs - Multiple Subject (Elementary Teacher), Single Subject (Middle School and High School Teacher), Education Specialist (Special Education Teacher), Integrated Teacher Education Program (ITEP - Education Specialist Credential), and Combined Credential and Master's Program - offered at Cal Poly Pomona. Please review the basic teaching credential program video and information below.

## Program Orientations

This orientation fulfills the program application requirement (item #10) and discussescredential coursework, credential pathways, clinical practice, teacher performance assessments (TPA), and next steps after admission.

## Education Specialist orientation

The Education Specialist orientation is available as an on-demand, self-paced virtual orientation. Please see the information at this [link](https://docs.google.com/forms/d/e/1FAIpQLScrxUb2WPMbCHBwYxX2U5JuzVCxjWlZMs3lB-Z0fnYSrk8Feg/viewform?usp=sf_link)to complete the orientation prior to applying to the program.

Please allow 5 business days to receive your completion certificate. Please email Dr. Heather Taylor, Education Specialist program coordinator if you have any questions or concerns about the virtual orientation.
