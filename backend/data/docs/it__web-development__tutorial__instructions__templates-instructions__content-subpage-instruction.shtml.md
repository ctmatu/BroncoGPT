# Content Subpage Tutorial

## Introduction

To start creating content for your website, go to the top left corner next to the site menu, and click on the Add Content button. Next, click on the Entity folder button and choose the [Content Sub Page](https://www-test.cpp.edu/green-and-gold/mockups/content-subpage.shtml) template.

![Image displays pictured steps to create add sub content with paragraph beforehand explaining steps.](../img/contentpage.png)

This checklist outlines each element of the new CPP web template for the content subpage. Use it as a guide to gather and organize all the necessary content for your page, ensuring you're well-prepared for a smooth content entry process in Cascade. **You can order the elements in any way you wish to best suit your page’s structure**.
