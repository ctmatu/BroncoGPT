# Submit a Housing Work Order

## Need to submit in a Housing Work Order?

Please use the online Work Order system to report any maintenance issue for your room, suite, building or the surrounding area around the residential buildings:

[Residents can submit a Work Order on the Housing Portal](https://cpp.starrezhousing.com/StarRezPortal) - (Requires CPP username and password)

Need help submitting a Housing Work Order? Check out our [How to Submit a Housing Work Order](https://www.youtube.com/watch?v=E02VL8gAqsM "Click to view video") video!
