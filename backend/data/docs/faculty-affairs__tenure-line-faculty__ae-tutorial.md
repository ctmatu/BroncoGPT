Fields with asterisks (\*) are required
 Please note that certain drop-down menus will cause the page to refresh.
 Please contact Faculty Affairs if you need assistance with the page.
  **Important:** Please make sure to verify that the faculty is not on leave or FERP before initiating the appointment.

| Field | Explanation |
| --- | --- |
| ![Form Type](img/form-types.jpg) | Select the appropriate Appointment type from the dropdown list.  Special Consultant Appointment -> for part-time faculty/ lectures (Job Code: 2358)  Faculty 2403- stipend -> Use this form if you are paying the faculty with a dollar amount.                                      TENURE LINE FACULTY (Job Code: 2360, 2482)                                      Full-time Lecturer (  **they should be teaching 15 units** ) (Job Code: 2359)  Faculty 2403 - WTU Base ->Use this form if you are paying using WTUs.  Other -> |
| ![Appointment Title](img/app-title.jpg) | Enter the Appointment Title. |
| ![Appointment Description](img/app-desc.jpg) | Enter the Appointment Description, please make sure that the number of characters does not exceed 1000 characters. |
| ![Department Code](img/dept-code.jpg) | Select the Department code from the dropdown menu, these department codes are associated with your colleges, if you don't find the appropriate code please contact HR. |
| ![Position Number](img/position-number.jpg) | Select the Position number from the dropdown, if you don't find the appropriate position number contact please contact Budget. |
| ![Chart Field String](img/cfs.jpg) | This value will be automatically filled, this is assigned to the position number, if the field is not automatically filled, please contact Budget. |
| ![Department Dean](img/dean.jpg) | Enter the Department Dean's name. |
| ![Appointment Period](img/app-period.jpg) | For the period of the appointment, please use the calendar icon to select dates, this will make sure that the date is in the correct format. |
| ![Appointment Dates](img/app-dates.jpg) | For the appointment date, first, select the term semester first. Second, select the month and year of the working dates, this will update the month grid.  Enter the number of working hours for each day, the working hours that are entered for weekends and holidays won't be calculated towards the assigned hours.  Click on the add button beneath the month grid, these hours will be shown in the below section.  If you wish to delete a day record, click on the delete button next to that record. |
| ![Total Hours](img/app-hours.jpg) | In this section, the total number of hours associated with the appointment will be automatically calculated after entering all the working dates in the previous section. |
| ![Appointment Salary](img/app-salary.jpg) | Enter the salary that will be paid to the member, and click on the calculate button to apply the formula for the TB Equivalent Fraction, Daily Rate, Monthly Rate, and FTE Monthly Rate. |
| ![Appointment Values](img/app-values.jpg) | These values will be automatically calculated after entering the Total Salary for the appointment. |
| ![Submit Appointment](img/submit-app.jpg) | Once the form is completed, click on Submit appointment. |

## Edit/Delete User

![Manage Users](img/manage-users.jpg)

 To Edit the access level to a specific user, click the Edit button next to the user record, this will prompt the user information to the top of the page, you can change the user access level in the access box.

To Delete the user from the list and block him/her from accessing the application click on the Delete button next to the user record.
