# Engineering Student Computer Accounts

#### [Return to College Computer Course Activation Index](http://www.cpp.edu/~engineering/departments/uums.shtml)

**What Is UUMS?**

1. 1. Any engineering student taking classes may acquire a user name and password to permit use of engineering computers.
   2. Once the student has activated their account they will have access to these computers with a courtesy storage quota of 50MB.
   3. If the student is enrolled in a class the instructor may activate printing and storage privileges as required by the course. A folder will be created with the name *MyEGRClassFolder* on the H:drive of the student's computer. Opening the folder will give access to each of the class folders for the courses in which the students is enrolled. However, class folders will NOT be created if the file storage quota has been exceeded.
   4. The student's user name for the College of Engineering computer account is the same their Cal Poly BroncoName. For the rest of the student's engineering career this user name will remain effective
   5. The password for the Engineering computer account can be set or reset using the following web page:
      <https://services.engineering.cpp.edu/PasswordRequest>

1. The student will be prompted to change the password at the first logon. The password must meet the following minimum requirements:
   1. Not contain the user's BroncoName.
   2. Be at least twelve characters in length.
   3. Contain at least one uppercase characters ( A through Z ) and one lowercase characters ( a through z )
   4. Contain characters of following categories:
      1. Decimal digits ( 0 through 9 ) or
      2. Non-alphabetic characters (for example, !, $, #, %)
2. Students must perform the first logon using an **Engineering Lab Computer** and choose a new password before using the *College Remote File Services Utility*.
3. A user will NOT have access to the *College Remote File Services Utility* if their password has expired.
