### Student Instructions for Requesting Selected Microsoft Products via MSDN Academic Alliance Software Center

Several Microsoft MSDNAA licensed products are available to Cal Poly Pomona College of Engineering students from the MSDN Academic Alliance Software Center. Some of the popular downloadable products include:

* + 1. MapPoint
    2. Project
    3. Visio
    4. Visual Studio
    5. Windows 7
    6. Windows 8
    7. Windows Server

**To request any or all of the above products:**

* + 1. Go on-line to the following URL: <https://services.engineering.cpp.edu/ELMSUIV/?action=signin>
    2. Click the "Log In" button and you will be redirected to our server for authentication.
    3. Read the User Agreement and click the “Agree” button at the bottom of the page to go to the next page.
    4. Enter your assigned Engineering Username and Password.
    5. Follow the on-line instructions to complete the request.
