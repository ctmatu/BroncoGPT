# About CalTPA

California requires all candidates for a preliminary Multiple, Single Subject, or Education Specialist Credential to pass a Teaching Performance Assessment (TPA).

The CalTPA includes two instructional cycles that measure the Teaching Performance Expectations (TPEs) and focus on the steps of *planning, teaching and assessing, reflecting and applying,* that naturally occur during teaching.

Cycle 1: Learning About Students & Planning Instruction/ Math Cycle

Cycle 2: Assessment-Driven Instruction/ Literacy Performance Assessment (LPA)

For more information, please visit the [California Educator Credentialing Assessments](http://www.ctcexams.nesinc.com/) website and select CalTPA under the Assessments’ tab.

**Alert**

Effective July 1, 2025, the Commission on Teacher Credentialing (CTC) will be transitioning to new CalTPAs. Candidates will complete these cycles in the Performance Assessment course while concurrently enrolled in Clinical Practice as follows:

|  |  |  |
| --- | --- | --- |
| **Credential Program** | CalTPA Cycle 1 | CalTPA Cycle 2 |
| **Multiple Subject** **EdSp- MMSN, ESN** | Math Performance Assessment – V1 | Literacy Performance Assessment (LPA) - V1 |
| **Single Subject** **World Languages** | Learning About Students & Planning Instruction – V1 | Assessment-Driven Instruction – V1 |

Further information about transition options is available in [PSA 25-03](https://www.ctc.ca.gov/docs/default-source/educator-prep/ps-alerts/2025/psa-25-03.pdf?sfvrsn=ebef3eb1_12) (see Table 1: Guidance for MS and ES Candidates or Table 2: Guidance for SS and WL Candidates) and [PSA 25-07](https://www.ctc.ca.gov/docs/default-source/educator-prep/ps-alerts/2025/psa-25-07.pdf?sfvrsn=b6543eb1_3) (Final Flexibilities for Meeting the MS and ES Candidate RICA Requirement).
