# Kurzweil Instructions

## About Kurzweil

Kurzweil 3000 text-to-speech software can help you read faster and improve your focus. You can use Kurzweil to read the web, including e-textbooks viewed through a browser.  You can also open almost any electronic file type with Kurzweil 3000, including Word, PDF, HTML, EPUB, and many others.

Anyone with a CPP email address can use Kurzweil for FREE!  With the campus license purchased by DRC, there are 3 ways you can use Kurzweil:

* Read web pages (including Canvas) using the **[Read the Web Chrome browser extension](http://support.cambiumtech.com/index.php?pg=kb.page&id=1597)**
* Use the [**web-based app**](https://www.kurzweil3000.com/)to read PDF, Word, and other document formats on any device connected to the internet

## Create your Free Kurzweil Account

1. Create a [free account with Kurzweil](https://accounts.kurzweil3000.com/fireflyweb/account/register.do?coordinator=Y3BvbW9uYTpTVFVERU5U) using your @cpp.edu address. (Note: External ID is not required.)
2. You will receive a confirmation email from Kurzweil Education Customer Service with your username and password, so **SAVE this email**!
3. Once you have your Kurzweil login, you are ready to use the web application ([kurzweil3000.com](http://www.kurzweil3000.com/)) from any device with a browser and an internet connection.

## How to Use Read the Web Extension

We recommend using Chrome browser extension to read the web (Firefox extension not recommended).

1. [Download and install the Read the Web extension from the Chrome webstore](https://chrome.google.com/webstore/detail/read-the-web/hdgegmlancchhhlkkddoiedlklgocffm)
2. Follow the prompts to install the extension.
3. After the extension is installed, restart Chrome.
4. You'll notice a color square icon button near the top-right of your web browser. You can click on that button to expand the Kurzweil overlay.
5. Enter your Kurzweil username and password to log into the Read the Web add-on.
6. Click the play button to start reading content on a web page. It may be a good idea to select text you want to read first, or place your cursor at a location on the page before you start to read.

Detailed [instructions on how to use Kurzweil Read the Web](https://www.kurzweiledu.com/kurzweil-academy/use-rtw.html) are available on the Kurzweil Academy page.

## How to Use the Web-based App

You can use [Kurzweil3000.com](https://www.kurzweil3000.com/) to read documents on any device with an internet browser and connection.

* You can open many different types of documents, including PDF, Word, and EPUB.
* You can upload and save documents in your Universal Library.
* You can also open documents from your computer or that are stored in Google Drive.
* For more information, please refer to [this 5-minute video](https://www.kurzweiledu.com/kurzweil-academy/video.html?id=375451029&tab=getting-started).

Detailed [instructions on how to use Kurzweil](https://www.kurzweiledu.com/kurzweil-academy/kurzweil-academy.html) are available from Kurzweil's Academy page.
