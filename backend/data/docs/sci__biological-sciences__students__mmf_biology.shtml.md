# Biology Markers, Milestones and Flexibility

## Biology – General Biology

#### Marker courses

* BIO 211/211L
* BOT 201/201L, MIC 201/201L, and ZOO 201/201L
* BIO 490

#### Milestones-Remain in the quarter system if the student:

* will be graduating by Spring 2020
* or, has taken at least four upper division (300- and 400-level) courses in the Required Option Core Courses

#### Flexibility in Quarter Curriculum

* Two out of the following: BOT 201/201L, MIC 201/201L, and ZOO 201/201L; use any 2000 or higher BIO course to substitute the missing units.
* Any upper-division Biology courses can be used to satisfy the Elective Support Courses.
* Use STA 1300 to substitute for BIO 211/211L
* Only one term of organic chemistry is required (CHM 2010/2010L or CHM 3140/L).
* Only one semester of biochemistry is required (CHM 3210 or 3270). No lab is required if a semester-long lecture is taken.
* Excess units for the all Option Core Courses can be used to satisfy Elective Support Courses; e.g., Form and Function in Plants (BIO 2050/2050L) is 4 total semester units and 6 quarter units; thus 2 excess units could be used for Electives. If Plant Physiology (BIO 4480/4480L) is taken as a semester-long course, there will be one extra unit that can be used for electives or to satisfy BIO 490
* Excess units for 400-level Option Core Courses (such as the physiology requirement) can be used for the Elective Option Core.
