# Program Info

## MDC Children Group Social

MDC provides 10-week Group Social program

* Day 1: Meet the teacher (MDC Social)
* Week 1: Assessment
* Week 2-4: Group-based Motor Skill Activities
* Week 5: Motor Skill Assessment
* Week 6-9: Group-based Motor Skill Activities
* Week 10: Assessment
* Week 11: Parent-Teacher conference

## MDC Children Program

MDC provides 10-week individualized motor skill program

* Day 1: Meet the teacher (MDC Social)
* Week 1: Present Level of Performance Assessment
* Week 2-4: Individualized Motor Skill Activities
* Week 5: Motor Skill Assessment
* Week 6-9: Individualized Motor Skill Activities
* Week 10: Present Level Performance Assessment
* Week 11: Parent-Teacher Conference

## MDC Adult Program

MDC adult program offers fitness-based adapted physical activity program.

* Week 1: Fitness Testing
* Week 2-9: Fitness Program Focus
  + Flexibility
  + Muscular Strength & Endurance
  + Cardiovascular Endurance
  + Balance
  + Adapted Sports
  + Lifetime Fitness
