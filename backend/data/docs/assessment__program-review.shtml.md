# Program Review

## What is Program Review?

Program Review is a systematic way to assess the quality of Cal Poly Pomona’s academic programs and determine ways to improve the quality of education, scholarship, and service. Internal and external reviewers provide an unbiased evaluation of the program to assure that CPP faculty and administration provide high-quality professional, graduate, and undergraduate programs for students, and to identify opportunities for improvement in each program.

Since the adoption of CPP’s new [Program Review Policy (PDF)](https://www.cpp.edu/academic-manual/1100-1199-curricular-policies/1100-1199/policy_1104_policy-on-program-review.pdf), MOUs have been signed to ensure action item-resource alignment. The graphic below provides a snapshot of the number of MOUs signed since the new program review policy was established.
