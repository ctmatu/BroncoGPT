## Applying to the Program

Join an existing cohort in Spring semester.

OR

Join a new cohort that will begin in Fall semester.

**Applications are due:**

November 1, 2024 for the Spring semester starting date.

OR

August 1, 2025 for the Fall semester starting date.

---

The Cal State Apply [application](https://www.calstate.edu/apply) for the Spring and Fall semesters is now open! Be sure to select if you are applying for the Spring semester 2025 to join an existing cohort or if you are applying for Fall semester 2025.

If you have questions about the Administrative Services Credential program, please email Mr. Jeff Seymour, Program Coordinator, at jjseymour@cpp.edu.

In addition to the Calstate apply application, a [supplemental application](https://engage.cpp.edu/register/admin_application) is required. For questions concerning supplementary application materials, contact Jeanne Reynaga, Credential Analyst at jmreynaga@cpp.edu.
