# STEM-Related CIP Code Changes

On occasion, departments may seek to change their program CIP code to enable their International students to qualify for the [STEM OPT Extension](https://studyinthestates.dhs.gov/sevis-help-hub/student-records/fm-student-employment/f-1-stem-optional-practical-training-opt). CIP changes are typically made effective at the beginning of a new academic year to ensure there is no disruption to student financial aid disbursement, international student visa considerations, and associated federal reporting requirements.

Programs that seek to change their CIP code to qualify for STEM designation can refer to **Chapter 5 in CPP’s [Curriculum Guide](https://www.cpp.edu/academic-programs/curriculum-development/2021-curriculum-guide-complete.pdf)** to learn more about the process.

**Contact the Curriculum and Scheduling staff within the [Office of Academic Programs](https://www.cpp.edu/academic-programs/about/index.shtml) for additional information.**
