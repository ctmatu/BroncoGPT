# Build a Garden

![seed packets for peas, radish, and sweet corn. Gardening tools](../img/build-garden.jpg)

## Build a Garden Develop new skills and learn about nature through gardening

The process of growing something from a seed or small plant can be fun. One of the great things about building a garden is that you don’t need a lot of space. For example, container gardening is a great way to control the soil, sun, and growing conditions of your plants.

![fun fact](../img/lightbulb.png)

We are more likely to try different fruits and vegetables when we grow them ourselves.
