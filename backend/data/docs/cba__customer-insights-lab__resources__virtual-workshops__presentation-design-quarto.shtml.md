# Stunning & Effective Quarto Slides with Reveal.js

## Designing Stunning & Effective Presentation Slides using Quarto

As part of CCIDM’s Spring 2025 Quarto Workshop Series, this session shows you how to turn plain slide decks into stunning, interactive Reveal.js presentations. You’ll learn to convert Quarto slides to formats like PPTX, add incremental lists and speaker notes, control visibility, and leverage engagement features such as navigation menus, overview mode, slide zoom, chalkboard, and multiplex. We also dive into theming: selecting built-in styles, customizing with SCSS variables, and crafting a brand-consistent look from scratch. By the end, you’ll know how to structure code blocks, manage content overflow, and present confidently—whether you’re teaching, pitching, or sharing data-driven stories.

**Learning Outcomes**

1. **Convert Revealjs presentation slides to other presentation formats**
2. **Build a dynamic and interactive presentation (e.g., incremental lists, speaker notes, multiple columns, and branded elements)**
3. **Maximize engagement with audience using various engagement-enhancing features of Quarto (e.g., navigation menu, overview mode, speaker view, and chalkboard)**
4. **Enhance presentation aesthetics by applying themes, customizing layouts, and incorporating animations or transitions**
5. **Customize themes using Sass variables to maintain brand consistency and create a distinctive presentation style**
