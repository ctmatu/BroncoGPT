# Define Phase Resources

## *![](../pictures/define_small.png)*

## *Define Phase: What problem would you like to fix?*

The tools and templates found here are used to identify your improvement opportunity and establish the goals of your project.

This table provides resources for the Define Phase of a continuous improvement project.

| **Template/Tool** | **User(s)** | **Description** |
| --- | --- | --- |
| [Project Charter (XLSX)](chartertemplate_v1.xlsx) | Project Leader, Project Champion & Project Participants | The facilitator, in consultation with the OE team and champion, will complete one of these at the beginning of every Continuous Improvement project. |
| [Stakeholder Analysis (XLSX)](stakeholderanalysis.xls) | Project Leader, Project Champion and Process Owners | This is a stakeholder analysis template that can be used in your project to identify key stakeholders and their influence on the project. |
