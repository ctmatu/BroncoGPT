# We are accepting applications

## Apply Now

Before you click the "Apply Now" button, please make sure you have all the application documents, indicated as below, ready. You may apply for multiple terms.

[Apply Now](https://cpge.cpp.edu//portal/applications/applicationProfile.do?method=loadApplicationIndex&applicationProfileId=1023089 "Apply Now. ")

![requirement laptop accepted message](../img/admissions-icon.jpg)

## Application Documents

1. Passport copy (the picture page)
2. Official Bank Statement (see below for requirements)
   1. Must be on official bank letterhead
   2. Contain accountholder name, account information, and account balance
   3. Statement needs to be in English and amount needs to be in U.S. dollars
   4. Must be dated within six months of desired program start date
   5. Please see chart below for minimum amount required -- based on how many terms you are applying to study
   6. If you are using someone else's bank statement, please have them fill out a [Statement of Financial Support Form](../docs/forms/statement-of-financial-support.pdf)
