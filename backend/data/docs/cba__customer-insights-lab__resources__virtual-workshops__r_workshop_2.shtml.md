# R Workshop #2: Data Types in R

## Data Types in R

Presented by Jarrod Griffin, Junior Marketing Scientist.
This workshop introduces the concepts of R data types and data structures. We cover numeric, logical, and character data types. We also cover R data structures such as vectors, matrices, data frames, and lists.

[If you haven't installed R or RStudio, click here to download on macOS or Windows!](https://www.cpp.edu/cba/customer-insights-lab/news/event/install-r.shtml)

**Learning Outcomes**

1. Explain atomic R data types - numeric, character, and logical.
2. Explain differences between data structures -- vectors, matrices, data frames, and lists.
3. Understand when to use each data type.
4. Create data frames.
5. Perform various operations to view and manipulate data frames.
