## Withdrawals

Students unable to attend courses offered through the College of Professional and Global Education after registering must request to withdraw from the course. Please follow the **[instructions on how to withdraw (PDF)](directions-for-students-to-withdraw-drop-a-course-accessible.pdf)**, and your refund will be processed automatically based on the timeline below. If a course is cancelled, students will receive a full refund for any tuition paid.

## Refund Timeline

Refund Timeline

| **Non Credit Classes: 8 or More Class Meetings** | |
| --- | --- |
| Prior to the 1st Class Meeting | 100% Refund |
| After the 1st Class Meeting | 65% Refund until 1/4 of the Class is Completed |
| After 25% of the Class is Complete | No Refund Available |

Refund Timeline

| **Non-Credit Classes: 4 -7 Class Meetings** | |
| --- | --- |
| Prior to the 1st Class Meeting | 100% Refund |
| After the 1st Class Meeting and before the 2nd Meeting | 50% Refund |
| After the 2nd Class Meeting | No Refund Available |

Refund Timeline

| **Non Credit Classes: 3 or Fewer Class Meetings** | |
| --- | --- |
| Prior to the 1st Class Meeting | 100% Refund |
| After the 1st Class Meeting | No Refund Available |

## Online (Asynchronous Courses) Courses

No refunds will be issued on or after the first day of class.
