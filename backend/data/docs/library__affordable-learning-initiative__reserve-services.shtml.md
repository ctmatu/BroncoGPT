# Using Reserve Services at the Library to Offer Alternatives to Expensive Textbooks

Reserve services are one of the resources to help faculty get started with the CSU's Affordable Learning Solutions program. This program is designed to reduce the cost of textbooks and other course related materials for CSU students, while at the same time, maintaining high academic standards.

* Reserve services will allow faculty to place books, DVD's, journals, etc. on limited loan for students to checkout.
* eReserve service will allow faculty to place sample tests, articles, lecture notes, homework solutions, and book chapters *online* for a specific course.
* These materials can be the faculty's personal copies or from the Library collections.
* Faculty can request the items to be place on permanent reserve or quarterly depending on the copyright guidelines.

Materials are displayed through the Library [**online catalog**](https://cpp-primo.hosted.exlibrisgroup.com/primo-explore/search?sortby=rank&vid=01CALS_PUP&lang=en_US) by searching for the instructors' last name or the course name or number.
