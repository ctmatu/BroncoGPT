# Project Detail

## A Deliberative Process

Cal Poly Pomona’s decision to develop the Lanterman property comes after a deliberative process that spanned several years, as the project timeline shows. The information in this section — the maps, reports and studies —document that process and underscore the university’s commitment to be transparent with campus and community constituents.
