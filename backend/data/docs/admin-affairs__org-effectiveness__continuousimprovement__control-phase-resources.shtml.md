# Control Phase Resources

## ![](../pictures/control_small.png)

## Control Phase: *How do you sustain the improvement?*

These tools and templates can help you with the implementation and control of your new process.

This table provides resources for the Control Phase of a continuous improvement project.

|  |  |  |
| --- | --- | --- |
| **Template/Tool** | **User(s)** | **Description** |
| [Control Plan (MS Excel)](processcontrolplantemplate.xlsx) | Process Owner | A method for documenting the functional elements of quality control that are to be implemented in order to make sure all standards are met. These should be updated as methods or measurements change. |
