## HEERA Manager JDXpert Resource Kit

The resources below apply Management Personnel Plan (MPP) employees WITH direct reports. If you are an MPP WITHOUT direct reports, please refer to the [Employee Resource Kit](resourcekits.shtml " Employee Resource Kit").

Follow the instructions below to launch the JDXpert resource videos.

1. For each video below, click on either the image or navigation link located above the blue button. This will open in a new tab and ask you to authenticate if you haven’t done so already.
2. Once authenticated, you will land to the course details page.
3. Follow the additional steps provided within the course details page.

[Frequently Asked Questions (FAQ)](https://cpp.service-now.com/employee_services?id=kb_search&kb_knowledge_base=4ac4e6b3db6e1810df9b72ec0f96197e&kb_category=dc320a9f87350e50b05164683cbb3580 "Frequently Asked Questions (FAQ)"): Access a repository of all JDXpert FAQs
