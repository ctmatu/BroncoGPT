# Scholarships

The Access and disABILITY Alliance awards financial scholarships each spring to Cal Poly Pomona students who have taken an active role in promoting disability awareness and serving as an advocate for people with disabilities. Applications for the Access and disABILITY Alliance may be submitted via [Cal Poly Pomona's Bronco Scholarship Application (BSA)](https://csupomona.academicworks.com/)
