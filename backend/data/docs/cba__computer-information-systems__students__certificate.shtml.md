# Certificates

The following CIS courses prepare students for professional certificates.

| **CIS Course** | **Professional Certificate** |
| --- | --- |
| CIS 2100 Programming for Business Analytics | Certified Entry-Level Python Programmer |
| CIS 2670 IT Infrastructure | AWS Cloud Partitioner |
| CIS 2900 Object-Oriented Programming for Business | Oracle Certified Junior Associate |
| CIS 3650 Web and Mobile Analytics | Google Analytics Certification |
| CIS 4670 Network Security | CompTIA Security+ |
| CIS 4810 Computer Forensics | Certified Forensics Examiner |
