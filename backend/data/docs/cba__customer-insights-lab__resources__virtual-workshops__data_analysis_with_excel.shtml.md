# Analyzing Data in Excel

## Data Analysis and Insights with Excel

This workshop is presented by the CCIDM’s Customer Relationship Management Specialist Joseph Lee on data analysis and insights using Microsoft Excel. In this video, he provides live tutorials on what cleaning, analyzing, and visualizing data looks like in Excel, shows how to clean and transform data into insights to create value using Excel tools, and aims to display the various real-world applications of the Excel functions and features.

[Click here for the Excel data file to follow along!](https://docs.google.com/spreadsheets/d/1nCMcY3kAvjX5DL3ewX7z8gaCUf-WbGP-/edit?usp=sharing&ouid=107261028077552979493&rtpof=true&sd=true)

**Learning Outcomes**

1. Recognize the basic data analytics cycle
2. Implement Excel Commands to clean, analyze, and visualize data
3. Apply new skills on various data sets
4. Execute analytical skills needed for almost any job (market researchers, data analysts, teachers/professors, managers, etc…)
