## How to set up your background

**Mobile: (size: 1080px by 1920px)**

iPhone

1. Download the image to your Photos.

2. Open Settings → Wallpaper.

3. Tap Add New Wallpaper.

4. Choose the image you saved.

5. Tap Set → Set Lock Screen, Set Home Screen, or Both.

Android

1. Download the image to your device.

2. Open the Photos/Gallery app.

3. Open the image → tap the More or Set as wallpaper option.

4. Choose Home screen, Lock screen, or Both.

**Desktop: (size: 1920px by 1080px)**

Mac (macOS)

1. Download the image.

2. Open System Settings → Wallpaper.

3. Click Add Photo or Choose Folder.

4. Select the image to set it as your desktop wallpaper.

Windows

1. Download and save the image.

2. Right-click the image file.

3. Select Set as desktop background.
