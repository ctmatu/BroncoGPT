# Funding Search

For your convenience we have provided easy to use databases using keywords to search for funding.

 Searchable Databases

[Pivot website](https://pivot.proquest.com/funding_main)
 Pivot is a global, web-based service that connects funding opportunity information, and researcher expertise. It will automatically email you funding opportunities matched to your keywords.

Please view our [workshop video](../training/index.shtml) on how to use Pivot

[Grant Resource Center](http://www.aascu.org/grc/gs/)

GRC's search tool is a customized database providing up-to-date funding information. Exclusively designed for public institutions to search and find federal and private grants information.

[Grants.gov](http://www.grants.gov/index.jsp)
 Grants.gov allows organizations to electronically find and apply for more than $400 billion in Federal grants. Grants.gov is THE single access point for over 1,000 grant programs offered by all Federal grant making agencies.

[The Federal Register](https://www.federalregister.gov/)

Access the Federal Register via GPO Gate, the University of California's gateway to federal information.
