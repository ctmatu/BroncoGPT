# Transportation

## Foothill Transit Silver Streak has arrived to CPP

CPP and Foothill Transit are engaged in a pilot program which offers Silver Streak Service to CPP. The Silver Streak stop is located in the SSB (Bldg 121) Turnout.

Service runs Mon - Fri from 7AM to 7PM.

There is no service on Holidays.

[Click here for a list of no service dates.](https://www.foothilltransit.org/holiday-schedules)
