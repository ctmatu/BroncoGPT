# Strategic Communications

## CPP This Week

![CPP This Week](img/cppthisweek-header1.png)

CPP This Week is a weekly e-newsletter sent to all faculty and staff on Tuesdays. It highlights announcements, events and stories about the university. The deadline to submit to CPP This Week is noon the Thursday before publication. Please send submissions to cppthisweek@cpp.edu. The maximum length is 120 words, with a headline that is four to eight words long. Please include a weblink, contact information, and a description, and if an event, include the date, time and location. Please also add events to the university’s [master events calendar](https://www.cpp.edu/events/). Announcements may be edited and will run a maximum of four weeks (please specify if fewer). Accuracy is the responsibility of the person submitting the information.
