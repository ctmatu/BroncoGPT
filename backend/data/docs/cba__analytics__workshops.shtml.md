# Workshops

The Center offered a series of workshops on analytics tools in partnership with student organizations. These workshops are designed to introduce students to in-demand software skills, and better prepare them for the workforce. Workshops offered in Fall 2020 included:

* [**Data Analytics Using R**](https://www.cpp.edu/cba/analytics/img/r-workshop-oct-29.png)
* [**Data Exploration and Visualization Using Tableau**](https://www.cpp.edu/cba/analytics/img/tableauworkshop.png)
* **[Location Intelligence Using Geographic Information Systems](https://www.cpp.edu/cba/analytics/singelyn-center-gis-workshops-sep-2020.pdf)**
