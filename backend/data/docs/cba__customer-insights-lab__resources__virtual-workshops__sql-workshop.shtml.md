# Introduction to SQL in Data Science

## Introduction to SQL in Data Science

The *Center for Customer Insights and Digital Marketing* presents Introduction to SQL for Data Science. The CCIDM’s Junior Marketing Scientist Christine Kuo will be leading the workshop. The workshop is designed to equip participants with essential SQL skills for basic data analysis. Participants will learn about SQL statement structure, how to write basic SQL statements, retrieve desired data from databases, and understand how to join and group tables. To be more familiar with how SQL works, we will offer participants hands-on practice to reinforce their data management and analysis concepts. This workshop will conclude with how SQL skills are related to  data visualization tools, the next step in data science workflow, and additional learning resources for participants. No preparation is required before the workshop.

**Learning Outcomes**

1. Describe SQL statement structure.
2. Identify key variables for data analysis.
3. Write basic SQL Statements to make queries.
4. Retrieve desired data from databases.
5. Explain the concept of joining and grouping tables.
