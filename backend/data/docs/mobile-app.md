## Cal Poly Pomona Mobile App

The CPP Mobile app is the official app from Cal Poly Pomona.

The app provides you with access to the latest information about Cal Poly Pomona wherever you are.

Use CPP Mobile to see your location on campus and locate buildings, view news, look-up class information, view campus events, follow Bronco Athletics, and more.
